# Canvas Design

Canvas Design 技能提供了完整的 Canvas 设计和绘图功能，帮助用户创建各种图形、动画和交互式视觉效果。

## 核心功能

- **图形绘制**：支持绘制线条、矩形、圆形、多边形等基本图形
- **路径操作**：创建复杂路径和形状
- **渐变和图案**：应用线性渐变、径向渐变和图案填充
- **文本渲染**：在 Canvas 上绘制文本，支持字体、大小和对齐方式设置
- **图像处理**：加载、修改和保存图像
- **动画效果**：创建平滑的动画和过渡效果
- **交互处理**：响应用户交互事件，如鼠标点击、移动等

## 使用场景

- **数据可视化**：创建图表、仪表盘和数据展示
- **游戏开发**：构建简单的游戏界面和元素
- **交互式演示**：创建动态演示和教程
- **创意设计**：制作艺术作品和视觉效果
- **用户界面**：构建自定义 UI 元素和控件

## 如何使用

1. **初始化 Canvas**：创建 Canvas 元素并获取绘图上下文
2. **设置样式**：配置线条宽度、颜色、填充等样式属性
3. **绘制图形**：使用 Canvas API 绘制所需的图形和元素
4. **添加动画**：使用 requestAnimationFrame 创建动画效果
5. **处理交互**：添加事件监听器处理用户交互
6. **优化性能**：合理使用 Canvas API，避免不必要的重绘

## 示例

```javascript
// 初始化 Canvas
const canvas = document.getElementById('canvas');
const ctx = canvas.getContext('2d');

// 绘制矩形
ctx.fillStyle = 'blue';
ctx.fillRect(10, 10, 100, 100);

// 绘制圆形
ctx.beginPath();
ctx.arc(150, 60, 50, 0, 2 * Math.PI);
ctx.fillStyle = 'red';
ctx.fill();

// 绘制文本
ctx.font = '20px Arial';
ctx.fillStyle = 'black';
ctx.fillText('Hello Canvas', 10, 150);
```

## 最佳实践

- **分层绘制**：将复杂图形分解为多个图层
- **缓存频繁使用的元素**：使用离屏 Canvas 缓存重复使用的图形
- **合理使用路径**：使用路径而非多个独立图形来提高性能
- **避免频繁重绘**：只在必要时更新 Canvas 内容
- **响应式设计**：根据容器大小调整 Canvas 尺寸

## 注意事项

- Canvas 是像素级绘制，不支持矢量缩放
- 大型 Canvas 可能会影响性能，需要注意优化
- 不同浏览器对 Canvas API 的支持可能存在差异
- 复杂动画可能需要使用专门的动画库来简化开发
